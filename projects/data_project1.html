Text Classification of Estonian Articles

About the project

This project involved the application of natural language processing and machine learning techniques to categorize Estonian articles. Various models, including Bag-of-Words, TF-IDF, and Word2Vec, were implemented for text classification. The project aimed to accurately classify articles into predefined categories ("www.advent.ee", "www.bioneer.ee", "www.lapsemure.ee", "www.naisteleht.ee", "arvamus.postimees.ee").
