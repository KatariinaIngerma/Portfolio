Website for beauty artist

About the project

Designed and developed a sleek and functional website for a permanent makeup artist. Leveraging HTML, CSS, and JavaScript, I crafted an engaging front page, a captivating portfolio section showcasing the artist's work, an easy-to-use contact page, and a transparent price list. This project demonstrates my proficiency in front-end development, user-centric design, and effective communication through web technologies.

Workflow

To ensure the website met the client's expectations, I began by thoroughly understanding their needs. After gathering requirements, I created wireframes to visualize the layout and structure. With a clear plan in place, I proceeded to implement the website using HTML, CSS, and JavaScript, focusing on delivering a sleek and functional user experience. Throughout the development process, I remained responsive to feedback, making necessary alterations to refine the design and functionality.
