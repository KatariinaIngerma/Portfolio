REST OpenAPI for downloading e-books.

About the project

This project involves building a REST API for various functionalities including downloading books from Gutenberg to the server, requesting book content, managing books, and searching for words within books. The REST API is implemented using the Python Flask framework, which offers a straightforward approach to creating RESTful web services. Additionally, an OpenAPI specification was created for the API, and server-side code was generated using the OpenAPI generator.
