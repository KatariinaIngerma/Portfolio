Tweet Sentiment Prediction

About the project

This project focuses on sentiment prediction for tweets using machine learning techniques. After preprocessing the dataset, which includes cleaning the text by removing numerical values, stop words, punctuation, mentions, and URLs, the sentiment column is mapped to numerical values: negative sentiment is represented as 0, neutral as 1, and positive as 2.

Two models are trained and evaluated: logistic regression and a Naive Bayes classifier. The logistic regression model achieves an accuracy of approximately 68.47% on the test data, while the Naive Bayes classifier achieves an accuracy of around 62.43%. These models enable the classification of tweets into one of three categories: negative, neutral, or positive, based on their sentiment.
